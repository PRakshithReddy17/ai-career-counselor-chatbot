# 🚀 AI Career Counselor Chatbot

An AI-powered career guidance chatbot that helps users explore career paths, get advice, and make informed decisions.

---

## 🌐 Live Demo

👉 https://ai-career-counselor-chatbot.vercel.app

---

## 🧠 Features

- 🤖 AI-based career advice
- 💬 Chat interface (like ChatGPT)
- ⚡ FastAPI backend
- 🌐 Deployed on Render & Vercel
- 🔐 Secure API handling (OpenRouter)

---

## 🛠️ Tech Stack

### Frontend
- Next.js
- React
- Tailwind CSS

### Backend
- FastAPI
- Python
- OpenRouter API (LLM)
